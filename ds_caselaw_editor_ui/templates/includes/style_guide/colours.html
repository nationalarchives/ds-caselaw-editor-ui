Colours

Primary colours

Usage details

The Yellow is used as an accent colour to give content emphasis and is used sparingly.
The Almost Black is used for headers and footers mainly to top and tail page layout.

The Aqua Blue and Dark Blue colours are used to signpost user interaction, for example links and buttons

Use these variables:

$color__yellow

Yellow

$color__almost-black

Almost Black

$color__aqua-blue

Aqua Blue

$color__dark-blue

Dark Blue

Secondary colours

Usage details

The grey suite of secondary colours are used for content panels to seperate and define page layout.

Use these variables:

$color__light-grey

Light Grey

$color__grey

Grey

$color__dark-grey

Dark Grey

Notification colours
