Notification messaging

Page notification

Used for page reload events – the message will stay on the screen until the user interacts with something else.

Use with class names page-notification--success or page-notification--failure:

Successfully signed in as Crispin.
The username and/or password you specified are not correct.

Context notification

Used for individual interactive elements such as forms which submit data via AJAX, when the message should be incorporated within the element itself, or placed nearby.

Use with class names context-notification--success or context-notification--failure:

Metadata has been saved successfully.
Please select at least one checkbox.
