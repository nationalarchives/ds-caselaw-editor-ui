Validation messaging

Page validation

Used for page reload events – the message will stay on the screen until the user interacts with something else.

Use with either .page-validation--success or .page-validation--failure

Successfully signed in as Crispin.

The username and/or password you specified are not correct.

Context validation

Used for individual interactive elements such as forms – these messages rely on JavaScript.

Use with either .context-validation--success or .context-validation--failure

Metadata has been saved successfully.

Please select at least one checkbox.
